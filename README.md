# 📸 Picsum Image Gallery

A simple and colorful image gallery built using HTML, CSS, and JavaScript, powered by the Picsum Photos API. It dynamically loads random images with a clean and responsive UI.

---

## Features
- Random images from Picsum API  
- Colorful modern UI  
- Responsive grid layout  
- Dynamic image loading  
- Smooth hover effects  

---

## Tech Stack
- HTML5  
- CSS3  
- JavaScript (Vanilla)  
- Picsum Photos API  

---

## Project Structure 
- index.html
- style.css
- script.js
---

## How to Run
1. Clone the repository  
2. Open `index.html` in your browser  
3. Click **Load Images**

---

## API Used
https://picsum.photos/
